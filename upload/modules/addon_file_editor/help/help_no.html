Admin verktøy: "Addon File Editor (AFE)"

Dette verktøyet lar deg se, redigere, slette, opprette, laste opp eller sikkerhetskopiere filer som tillhører Tillegg som for eksempel moduler, designmaler og språkfiler fra Website Baker Admin sidene uten å laste ned disse filene. Du kan også lage installasjonspakker av innstallerte Tillegg slik at disse kan installeres via Website Baker Admin sidene under Tillegg.

A: Installasjon

  1. Last ned "addon_file_editor_xx.zip" og inslaller via Website Baker Admin sidene under: -> Tillegg -> Moduler
  2. Gå til verktøyet via Website Baker Admin sidene under: -> Administrasjonsverktøy -> Addon File Editor.

B: Krav og begrensninger

Addon File Editor krever PHP 4.3.11 eller høyere. Siden AFE v0.80, er Website Baker 2.8 (RC2 eller høyere) påkrevet for å kunne benytte alle funksjoner, slik som jQuery Toggle eller "Syntax Highlighting". AFE v0.80 fungerer med WB 2.7, men da med begrensede funksjoner. For å slå på "syntax highlighting" i AFE 0.80 for WB 2.7, følg instruksjonene i sksjonen tilpasset modus. WB 2.7 brukere kan eventuelt fortsette å benytte versjon AFE v0.71 (ikke lenger vedlikeholdt).

Addon File Editor støtter ikke WB 2.6.x eller lavere.

Kun tillegg som fyller kravene under kan redigeres med Addon File Editor:

  1. tillegg må være oppført i WB databsen under "addons" (installert via WB)
  2. tillegg må være lesbre for PHP
  3. det må være en index.php fil i tilleggets rotfolder
  4. for å redigere tillegg som er merket med rødt, må du først aktivere FTP funksjonen (se seksjon D)

Tilpasset modus (WB 2.7 + AFE 0.80):
Det er to ting som er påkrevet, for å aktivere "syntax highlighting" for Website Baker versjon 2.7 og AFE 0.80.

  1. kopier /include/editarea folderen fra en WB 2.8 installasjonspakke til folderen /include i WB installasjonen din
  2. døp om filen /modules/addon_file_editor/backend_legacy.js til backend.js

OBS:
Addon File Editor 0.71 var den siste versjonen som ble utvikklet for Website Baker platformen 2.7. For å dra nytte av alle poensialene i dette administrasjonsverktøyet, kreves det at du benytter Website Baker 2.8 eller høyere.

C: Jobbe med Addon File Editor

Over sikt over Tillegg:

Verktøyet lager en liste over alle installerte Tillegg, når du strater dette via: -> Administrasjonsverktøy -> Addon File Editor.

Tilleggene er grupper i følgende seksjoner: Moduler, Designmaler og Språkfiler. Hvis du har aktivert Javascript, vil seksjonene automatisk starte som et minimert filtre. Du kan maksimere/ minimere seksjonens filtre med et museklikk. Tilstanden for filtreet i en seksjon, lagres ved å sette en innformasjonskapsel på din maskin, og vil huskes så lenge denne innformajonskapselen er på maskinen din.

For å åpne folderen for et spesifikt Tillegg i filbehandleren, klikke på navnet på Tillegget. For å lagen en sikkerhetskopi av et installet Tillegg med alle tillhørende filer og foldere, klikker du på nedlastningsikonet til høyre i listen.

Filbehandleren:

Filbehandleren viser filene og folderene for et spesifikt Tillegg. Som standard, vises kun filer med endelse som kan gjenkjennes (tekst, bilder, arkiver). Du kan endre hvilke filendelser som gjenkjennes ved å redigere config.inc.php. For å vise alle filendelser, redigerer du parameteret $show_all_files i config.inc.php.

For å redigere en tekstfil, eller se et bilde i et nytt nettleservindu, klikker du på filnavnet. For å dra nytte av syntax highlighting, er det påkrevet med Website Baker 2.8 eller høyere og Javascript. For å "omdøpe" eller "slette" filer, klikker du på ikonet for behandling som finnes til høyre i filbehandleren.

Du kan lage nye filer/ foldere eller laste opp filer via linkene som vises på toppen av filbehandleren. Oppdater funksjonen tvinger filbehandleren til å laste inn alle filer og foldere på nytt. Benytt denne funksjonen hvis du har installert et nytt tillegg.

Hvis du har slått på støtte for Pixlr (i config.inc.php filen), vises også et ikon for redigering av bilder. Klikk på dette ikonet for å sende et bilde fra ditt nettsted til online bilderedigeringsverktøyet Pixlr for redigering der. Når du lagrer endringer du har gjort i Pixlr, vil bildet bli lastet ned til nettstedet dit og omdøpt fra bilde_navn.jpg til bilde_navn.pixlr.jpg. Det orginale blir derfor ikke berørt av denne redigeringen (du kan slette dette hvis det ikke lenger er behov for dette). For å benytte deg av Pixlr tjenesten må du ha installert en Flash Plugin i nettleseren din. For fler detaljer se Kapittel D.

D: Innstillinger og Konfigurasjon

De fleste innstillinger kan endres via konigurajonsfilen config.inc.php som du finner i Addon File Editor folderen. Du kan føye til eller fjerne filtyper for tekst, bilde og arkiv filer. Følgende filtyper gjenkjennes som standard filtyper:

  • $text_extensions: 'txt', 'htm', 'html', 'htt', 'tmpl', 'tpl', 'xml', 'css', 'js', 'php', 'php3', 'php4', 'php5'
  • $image_extensions: 'bmp', 'gif', 'jpg', 'jpeg', 'png'
  • $archive_extensions: 'zip', 'rar', 'tar', 'gz'

Filtyper med andre endelser en det som er listet opp over er normalt satt til å være skulte ($show_all_files = false). For å skjule bestemte Tillegg, legger du til stien til folderen som Tillegget ligger i under tabellen $hidden_addons = array(). Hvis du legger til 'addon_file_editor" i denne tabellen, vil du ikke lenger ha mulighet for å endre config.inc.php filen i AFE.

Den maksimale størrelsen for opplasting er begrenset til 2 MB ($max_upload_size = 2), Pixlr støtte er som standard slått av ($pixlr_support = false).

Verktøyet har integrert en valgfri FTP støtte, hvilket gir deg tilgang til Tillegg og filer som vaneligvis eies av ftp-user (slik som WB kjerne designmaler eller kjerne moduler som er lastet opp med FTP). Du kan konfigurere FTP støtte med FTP assistenten via: http://yourdomain.com/modules/addon_file_editor/ftp_assistant.php
Filer som kun kan redigeres med FTP støtte påslått er markert med rødt.

E: Pixlr Bilde Redigering (FAQ)

For å kunne redigere bilder, er tredjeparts online fororedigeringsverktøyet http://pixlr.com benyttet. For å kunne benytte Pixlr, må flølgende alle av de følgende gravene være oppfylt:

  1. Du må aktivere denne funksjonen manuelt ved å endre følgende parameter til$pixlr_support = true; i(config.inc.php)
  2. Det kreves at nettleseren din har installert Adobe Flash Player Plugin
  3. Bildet som skal redigeres må være tilgjengelig på webserveren (bilder på din egen maskin vil ikke kunne redigeres)
  4. PHP må ha både lese og skrivetilgang til bildet (det er foreløpig ikke støtte for FTP)
  5. Følgende bildeformater støttes: Lese (gif, jpg, png); Skrive: (jpg, png)

OBS:
Bildet du redigerer blir lastet opp til pixlr.com og slettes automatisk etter 5 minutter. Les ellers Pixlr FAQ før du benytter deg av denne tjenesten.

F: Feilretting

Når du starter Addon File Editor, vil den gjeldende statusen for alle installerte Tillegg bli analysert og lagt i en hurtigbuffer. Dette gjør at du kan benytte Addon File Editor uten å måtte gnerer liste over Tilleggene på nytt. Dette har en ulempe. Hvid du installerer eller avistalerer et Tillegg i fra Admin sidene i Website Baker etter at File Editor ble startet, må du OPPDATERE listen over Tilleggene, slik at endringene på serveren blir synlige i listen. Dette kan gjøres ved å klikke på [ Oppdater ] linken på toppen av oversiktslisten eller på toppen av Filbehandleren. Hvis du skulle støte på problemer med å oppdatere listene, prøv å tømme, "cache" i nettleseren din, innformasjonskappslene (cookies) og de lagrede sesjonsvariablene (f.eks. ved å logge ut), og så prøve på nytt.
